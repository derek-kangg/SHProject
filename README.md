# FORME — AI Personal Stylist

Clean, minimal AI styling website. Next.js 15 · TypeScript · Tailwind CSS.

## Setup

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Pages

| Route | Feature |
|---|---|
| `/` | Today — daily outfit suggestions |
| `/closet` | My Closet |
| `/builder` | Style Builder — build around a piece |
| `/saved` | Saved Looks |
| `/tryon` | Avatar Try-On |
| `/shop` | Smart Shopping |

## Collaboration

- Branch per feature: `git checkout -b feat/<name>`
- Never push directly to `main` — always open a Pull Request
- Teammate assignment: see README section below
